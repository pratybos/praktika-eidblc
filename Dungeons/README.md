# Dungeon Explorer

Tai yra paprastas konsolinis C++ turn-based/RPG tipo žaidimas.
Žaidėjas keliauja po atsitiktinai sugeneruotus lygius, kovoja su atsitiktinais priešais, renka daiktus, atsirakina pasiekimus, pasikelia lygį t.t.

Žaidimo tikslas - išgyventi (neprarasti HP), surinkti visus galimus lobius ir pasiekti išėjimą žymimą `X`

### Žemėlapio žymėjimas:
`#` - siena;
`.` - grindys;
`P` - žaidėjas;
`E` - priešas;
`T` - lobis;
`S` - parduotuvė;
`X` - išėjimas/pabaiga;

### Žaidimo valdymas:

```text
W = judėti aukštyn
A = judėti į kairę
S = judėti žemyn
D = judėti į dešinę
I = atidaryti inventorių
H = atidaryti pasiekimų sąrašą
Q = grįžti į pagrindinį meniu
